#Codebook For UCI HAR Dataset run_analysis.R

###Source Data Original Data, Full Description

###Data Transformation Steps

  • Read Activity Label, Mean/Std Columns and Set Data x_*.txt for Test Data Set.
  • Append ActivityId activity_ label.txt and Subject Id subject_test.txt.
  • Subset mean and std value columns.
  • Read Activity Label, Mean/Std Columns and Set Data x_*.txt for Training Data Set.
  • Append ActivityId activity_ label.txt and Subject Id subject_train.txt.
  • Subset mean and std value columns.
  • Merge Train and Test data sets.
  • Add ActivityName column to merged data set.
  • Melt merged datasets with id variables ActivityId, ActivityName, Subject ID and measure variable with mean and std values.
  • DCast melted dataset with mean aggregation function over activity name and subject id for all variables
  • Write output file tidy.txt in working directory

###Variable Descriptions Detailed description in featuresinfo.txt, README.txt files.
